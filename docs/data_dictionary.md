# Data Dictionary

This file should be kept in sync with `etl/transform.py` and the Power BI model.

Example table: sales
- sale_id: string — unique identifier
- date: date — transaction date
- product_id: string — product key
- quantity: integer — units sold
- revenue: float — sale amount in USD
